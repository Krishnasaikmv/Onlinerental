# 🚗 Online Rental System (PHP & MySQL)

A dynamic **Online Rental System** developed using **PHP and MySQL** as part of my Web Development Internship (Task 2).

This project demonstrates a complete **CRUD application** with user authentication and image upload functionality.

---

## 📌 Features

- 🔐 User Registration & Login
- 🔒 Secure Password Hashing
- 👤 Session Management
- ➕ Add Rental Items
- 👀 View Rental Items
- ✏️ Edit Rental Items
- ❌ Delete Rental Items
- 🖼️ Image Upload Functionality
- 🎨 Attractive User Interface

## 🚀 How to Run the Project

1. Install **XAMPP**
2. Start **Apache** and **MySQL**
3. Place the project folder inside: C:\xampp\htdocs\
4. Open **phpMyAdmin**
5. Create a database named: blog
6. Create required tables (users & posts)
7. Open browser and run: http://localhost/OnlineRental/Task2-OnlineRentalSystem/
